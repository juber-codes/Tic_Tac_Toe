# 🎮 Tic-Tac-Toe Game with AI (Python + Tkinter)

A simple and interactive **Tic-Tac-Toe game** built using Python.
This project supports both **Multiplayer mode (2 players)** and **Single-player mode with AI 🤖**.

---

## 🚀 Features

* 🎯 **Two Game Modes**

  * Player vs Player
  * Player vs AI

* 🤖 **AI Opponent**

  * Smart AI using logic-based decision making
  * Multiple difficulty levels (Easy / Hard)

* 🎨 **Graphical User Interface**

  * Built using Tkinter
  * Clean and user-friendly design
  * Colorful buttons and game board

* 🧠 **Game Logic**

  * Win detection
  * Draw detection
  * Score tracking

---

## 🛠️ Technologies Used

* **Python 3**
* **Tkinter (GUI Library)**
* **Random Module (for AI moves)**

## 🎮 How to Play

* The game is played on a **3x3 grid**
* Player 1 is **X**, Player 2 / AI is **O**
* Players take turns placing their marks
* First to get **3 in a row (horizontal, vertical, diagonal)** wins
* If all cells are filled → it's a **Draw**

---

## 🤖 AI Logic

* **Easy Mode:** Random moves
* **Hard Mode:** Smart moves (tries to win or block opponent)
